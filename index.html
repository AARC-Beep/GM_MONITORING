SeaCrew Dashboard — Daily Routine Log

This help contains login, usage, and troubleshooting steps.
Login: Username & password from the Users sheet. Roles: admin (can edit/delete) and user (view + chat).

  1. Login with your credentials.
  2. Use the sidebar to open sections. Add entries using the form on top of each section.
  3. To update an existing entry: press the pencil icon — the form will populate — press Update.
  4. To delete: press trash — only admins can delete. Deletions are recorded server-side and monthly archive keeps snapshot.
  5. Chatboard: anyone can post. New messages play a bell sound.
