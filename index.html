User Guide

This dashboard syncs to your Google Sheet web app. Login with your Username/Password from the Users sheet. Admins can edit/delete records. Users can add records only.

  1. Click Login → enter Username and Password (stored in Users sheet).
  2. Use the left sidebar to navigate between sections.
  3. After adding a record, it will attempt to sync automatically. Use "Sync now" if needed.
  4. To remove a row from the Google Sheet, delete it from the entry (admin only). Deletion calls the web app delete endpoint.
